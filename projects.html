Improving Adherance to Medical Treatments - BMEG 357
Jan 2023 - Apr 2023
Skills: C, Python, Plotly, HTML, JavaScript, CSS, Engineering, Medical Engineering, Electrical Engineering, Microcontroller

This project was presented by Dr. Andrew Thamboo at St. Paul’s Sinus Centre in British Columbia during my studies, and focused on addressing patient noncompliance with post-operative treatment for chronic sinus disease.

The final design concept involved both hardware and software components. The hardware included a button attached to the sinus rinse bottle, soldered onto a microcontroller. This board communicated via Bluetooth with the users' phones, enabling data logging, encryption, and trend visualization on an app and web based dashboard.

The logged data was stored in a central database, allowing the app to provide reminders and positive reinforcement to encourage adherence. The data also enabled the creation of a dashboard for facilitating data visualization and analysis for doctors, allowing them to search and interpret patient data with ease, and ultimately develop personalized treatments.

This project was a cooperative team effort. I was incharge of the circuitry, communication between different components of the design, building the central database, populating the database with data for testing, and the doctors dashboard.

Solidworks was employed for 3D printing and prototyping the hardware casing. The hardware system included a microcontroller circuitry based on an Adafruit HUZZAH32 – ESP32 Feather Board. The code for controlling the microcontroller was written in C. On the software side, the patient user interface application was built using React.js, incorporating HTML, CSS, and JavaScript. The doctors dashboard was created using Plotly Dash. The data storage relied on google cloud services. The scripts to create data and populate the database were written in Python.
